# GitHub About 文案包（中文）

## About（仓库简介）

### 一句话简介（Short description）

面向资本市场的开源事件智能终端：因果分析、路径推演、市场映射与回放验证。

### 两句话简介（Long description）

HEM-CM 把突发新闻与热点事件转成可解释的因果链、阶段感知的多情景预测，以及跨市场影响映射。  
它同时提供可回放的研究工作流，用于验证模型与复盘事件驱动的市场走势。
